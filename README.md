# 🍽️ HOTEL TRIMUKHA — Restaurant Website
A modern, mobile-friendly restaurant website for **Hotel Trimukha**, built with pure HTML, CSS, and JavaScript. Features a rich warm color theme, image slideshow, video hero section, chef's specials, and WhatsApp ordering integration.
## 🌐 Live Preview
> Deploy on GitHub Pages to get a live link (see [Deployment](#-deployment) section below).
---
## ✨ Features

- 🎠 **Auto-sliding Hero Carousel** — Showcases restaurant ambiance with smooth image transitions
- 🎬 **Video Hero Section** — Full-screen looping background video for immersive experience
- 🍛 **Chef's Specials Section** — Highlighted dishes with hover animations
- 📱 **Fully Responsive** — Mobile-first design with hamburger menu & bottom navigation bar
- 💬 **WhatsApp Order Button** — Direct link to place orders via WhatsApp
- 🎨 **Royal Warm Theme** — Deep red, burnt orange, and golden color palette using CSS gradients
- 🔗 **Multi-Page Navigation** — Links to Menu, About, Gallery, Contact pages

## 🚀 Deployment

### Option 1 — GitHub Pages (Recommended)

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```

### Option 2 — Run Locally

Simply open `index.html` in any modern browser — no server or installation needed.

```bash
# Clone the repo
git clone https://github.com/<your-username>/hotel-trimukha.git

# Open in browser
cd hotel-trimukha
open index.html
```
